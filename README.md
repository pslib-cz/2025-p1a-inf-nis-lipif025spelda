Prumkaskvelyobed.cz

Informační systém, který ukáže školní obědy průmyslové střední školy v Liberci. Uživatelé mohou komentovat a ohodnotit zkušenost.

## Perex

Informační systém prumkaskvelyobed slouží studentům a zaměstnancům SPŠ a VOŠ Liberec k transparentnímu hodnocení školního stravování. Systém každý den (pondělí–pátek) zobrazuje trojici podávaných obědů s fotografií, datem a číslem menu. Uživatelé mají možnost jídla okamžitě oznámkovat palcem nahoru/dolů, případně připojit slovní komentář.
